# WutToDo - Tasks App

Creating my own To-Do Tasks web application while learning Web Development. The app is being updated incrementally as I learn new concepts and technologies.

## Features

- Add tasks
- Delete tasks
- Mark tasks as completed
- Live statistics dashboard
- Enter key support

## Changelog

## Development History (Latest to Oldest)

### v0.8.0 Added local storage support
- Added browser-based task saving using localStorage
- Tasks are automatically restored after refreshing the page
- Saved task completion status

### v0.7.0 - State-Based Architecture
- Migrated task storage from DOM elements to an array of task objects
- Introduced centralized rendering with renderTasks()
- Implemented state-driven UI updates
- Improved maintainability and scalability

### v0.6.0 - Core Task Management Features
- Added live statistics dashboard
- Refactored JavaScript into a separate file
- Improved project structure

### v0.5.0 - Task Completion
